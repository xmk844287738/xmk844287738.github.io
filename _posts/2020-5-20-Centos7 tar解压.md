### 解压 tar包
```
tar -xvf file.tar 
```

### 解压tar.gz
```
tar -xzvf file.tar.gz 
```

### 解压 tar.bz2
```
tar -xjvf file.tar.bz2 
```

### 解压tar.Z
```
tar -xZvf file.tar.Z 
```

### 解压rar
```
unrar e file.rar 
```

### 解压zip
```
unzip file.zip 
```
